## Project Description

A task management application using Clean Architecture and DDD (Domain-Driven Design) patterns.

### Tech stack

- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **Testing**: Jest, Supertest

### Project launch

```bash
npm install
npm run dev
```

### Testing

```bash
npm run test:unit
npm run test:int
```
